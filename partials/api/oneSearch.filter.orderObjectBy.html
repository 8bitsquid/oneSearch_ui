orderObjectBy
filter in module oneSearch

Description

Order's an array of objects by the value of a property in those objects.

Usage

In HTML Template Binding

{{ orderObjectBy_expression | orderObjectBy:field:reverse }}

In JavaScript

$filter('orderObjectBy')(items, field, reverse)

Parameters

ParamTypeDetails
itemsArray.<Object>

An Array or Objects to order

fieldstring

The field/property of the objects to order by

reverseboolean

Order objects in reverse

(default: false)

Example

var arrObj = [
     {n: 10},
     {n: 1},
     {n: -5}
 ];

 var ordered = $filter('orderObjectBy')( _engines, 'n');
 /*
   ordered is given an array of objects, ordered by `n`:
   [
     {n: -5},
     {n: 1},
     {n: 10}
   ];
 */
