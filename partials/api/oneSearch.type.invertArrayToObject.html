invertArrayToObject
function in module oneSearch

Description

Given an object with an array as it's value, this function will create a new object having the array keys as separate object keys and the old object keys as their value

Usage

new invertArrayToObject(obj);

Parameters

ParamTypeDetails
objobject

An Object with an Array as the property values

Returns

object

An new, inverted object, where the values of the Array are now the properties assigned a value of the old property

Example

var myObj = {
     field: ['value1', 'value2']
};
var invertedObj = invertArrayToObject(myObj);
// The resulting invertedObj will look like:
// {
//   value1: 'field',
//   value2: 'field',
// }
