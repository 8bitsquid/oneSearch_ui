toArray
function in module oneSearch

Description

Converts the given object into an Array of it's values

Adopted from https://github.com/a8m/angular-filter/blob/master/src/_common.js

Usage

new toArray(object);

Parameters

ParamTypeDetails
objectobject

Object to convert into an Array.

Returns

Array

Array of the object's property's values

Example

var myObj = {
 prop1: "Don't",
 prop2: "cross the",
 prop3: "streams!"
};

var arrayNowPlease = toArray(myObj); // returns ["Don't", "cross the", "streams"]
