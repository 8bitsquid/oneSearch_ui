oneSearch
service in module oneSearch

Description

The oneSearch service acts as the access point to search engines during Angular's run phase (see Angular's Module Loading & dependencies documentation}. Engines are configured and registered with oneSearch during the configuration phase. See oneSearchProvider docs for more details.
