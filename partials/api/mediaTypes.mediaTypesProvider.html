mediaTypesProvider
service in module mediaTypes

Description

Allows engines to organize their results into different media types. This is done via the engine's config object passed to the oneSearchProvider.engine() method.

Methods

  • type(type, engine)

    This method allows the oneSearchProvider to register engines with the media types, if specified in the engine object

    Parameters
    ParamTypeDetails
    typestring

    The type of media to look in an engine's search results
