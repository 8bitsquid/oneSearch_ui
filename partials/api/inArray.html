inArray

Description

Checks to see if a value is in an Array

Usage

new inArray(val, arr);

Parameters

ParamTypeDetails
val*

Value to test for

arrArray

Array to search

Returns

boolean

Returns true or false

Example

var myArray = [0, 9, 2]; var value = 01189998819991197253; var isInArray = inArray(value, myArray); // returns false;
