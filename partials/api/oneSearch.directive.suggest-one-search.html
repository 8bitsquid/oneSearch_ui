suggest-one-search
directive in module oneSearch

Description

Directive for the search box with suggestion dropdown

Dependencies

  • $timeout
  • $document

Usage

as element:
<suggest-one-search
       prompt="{string}"
       model="{string}"
       search="{string}">
</suggest-one-search>
as attribute
<ANY suggest-one-search
     prompt="{string}"
     model="{string}"
     search="{string}">
   ...
</ANY>
as class
<ANY class="suggest-one-search prompt: {string}; model: {string}; search: {string};">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
promptstring

The placeholder text to display in the search text box

modelstring

Name of the $scope variable to store the search string

searchstring

Name of the $scope reference to the search() function defined in the oneSearch.oneSearch:oneSearchCtrl
